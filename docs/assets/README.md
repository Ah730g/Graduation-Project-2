# مرفقات وثيقة التخرج (Assets)

هذا المجلد مخصص للصور/المخططات التي يصعب تمثيلها نصيًا داخل Markdown (مثل UML Use Case Diagram، Activity Diagram، Wireframes، ولقطات دليل الاختبار).

## هيكل مقترح

- `usecase.png`: مخطط حالات الاستخدام العام (UML).
- `activity_add_post.png`: Activity Diagram لتدفق إضافة إعلان (Draft/Publish).
- `activity_contract_signing.png`: Activity Diagram لتدفق الدفع→العقد→التوقيع.
- `wireframe_add_post.png`: Wireframe لواجهة إضافة إعلان (اختياري).
- `screenshots/`: لقطات شاشة (Evidence) لنتائج الاختبار.

## أدوات مقترحة للإنتاج

- **draw.io (diagrams.net)**: مناسب لـ UML/Activity/Component/Deployment.
- **StarUML / Visual Paradigm**: مناسب لـ UML رسمي (Use Case/Activity/Sequence/Class).
- **Figma**: مناسب لـ Wireframes و UX flows.
- **Windows Snipping Tool**: لالتقاط الأدلة (Screenshots) أثناء الاختبار.
